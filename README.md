# 🎨 Color Generator

🎨 Gerador de cores em JavaScript

Projeto desenvolvido com o objetivo de praticar lógica de programação e manipulação do DOM.

---

## 🚀 Funcionalidades

* 🎲 Geração de cores aleatórias (HEX)
* 📋 Clique para copiar o código
* ⚡ Atualização dinâmica na tela

---

## 🧠 Conceitos praticados

* Manipulação do DOM
* Funções
* Geração de números aleatórios
* Eventos de clique
* Criação dinâmica de elementos
* Estilização com CSS

---

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript

---

## 📌 Objetivo do projeto

* Reforçar fundamentos do JavaScript
* Praticar lógica de programação
* Trabalhar com elementos dinâmicos na interface
* Evoluir um projeto simples

---

## ▶️ Como executar

Clone o repositório:

```
git clone https://github.com/seu-usuario/color-generator.git
```

Abra o arquivo `index.html` no navegador
